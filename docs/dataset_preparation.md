# Dataset Preparation

Datasets Used

- CASIA-A
- CASIA-B
- OU-ISIR

Preparation Steps

1. Download dataset.
2. Extract video sequences.
3. Convert videos into frames.
4. Resize images.
5. Apply AdLOG-WF preprocessing.
6. Generate segmented silhouettes.
7. Prepare train and test partitions.
8. Train classification modules.
