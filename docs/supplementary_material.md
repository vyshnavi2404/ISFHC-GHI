# Supplementary Material

## Hybrid Gait-Based Human Identification Using Modified U-Net and Score-Level Fusion

### Overview

This repository accompanies the ISFHC-GHI framework proposed for gait-based human identification.

The repository provides representative implementations of:

- Adaptive LOG-Based Wiener Filtering (AdLOG-WF)
- Modified U-Net Segmentation
- Hierarchy of Skeleton Feature Extraction
- Multitexton Feature Extraction
- Color Feature Extraction
- LinkNet Classification
- SqueezeNet Classification
- Improved Score-Level Fusion
