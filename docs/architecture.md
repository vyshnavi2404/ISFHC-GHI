# ISFHC-GHI Architecture

The proposed framework integrates:

1. Adaptive LOG-Based Wiener Filtering

2. Modified U-Net Segmentation

3. Hierarchy of Skeleton Feature Extraction

4. Multitexton Feature Extraction

5. Color Feature Extraction

6. LinkNet Classification

7. SqueezeNet Classification

8. Improved Score-Level Fusion

The framework combines structural,
textural, appearance-based, and
deep-learning representations for
gait-based human identification.
